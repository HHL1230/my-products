## 📦 產品列表

### 1. ⚖️ Scale Monitor (天平數據傳輸程式)
專為實驗室設計的自動化工具。透過 RS232 直連技術，將 Mettler Toledo 與 OHAUS 等天平的重量數據精準、穩定地導入 Excel，徹底解決手動輸入的失誤與輸入法干擾。
- **特點**：背景寫入、防連發機制、多方向游標移動。
