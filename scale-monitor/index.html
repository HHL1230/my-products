Scale Monitor

Version 1.1.0 - Stable

專為實驗室設計的自動化工具。透過 RS232 直連技術，將天平重量數據精準、穩定地導入 Excel，告別手動輸入與輸入法干擾。

輸入法避讓技術

背景寫入技術

即使 Excel 視窗不在焦點（正在處理其他工作），數據依然能準確寫入指定儲存格。

智慧自動化

支援多方向游標移動、智慧閒置斷線，完美契合 LIMS 系統作業節奏。

📖 快速上手指南

1. 安裝與準備

  • 請將下載的 ScaleMonitor.exe 放置於您的桌面（或任意方便使用的資料夾）。
  • 確保天平已透過 RS232 連接線或 USB 轉 COM 線正確連接至電腦。
  • 確認電腦中已安裝並可正常開啟 Microsoft Excel。

2. 首次啟動與配置

第一次啟動程式時，畫面會列出目前偵測到的通訊埠。請依據終端機畫面提示，設定以下參數（若不修改直接按 Enter 即可使用預設值）：

  1. COM Port：輸入天平連接埠的編號（例如 COM1）。
  2. RS232 通訊參數：包含 Baud Rate (預設 9600)、Parity、Data Bits、Stop Bits。必須與您天平內部的設定值完全一致。
  3. 下一格移動方向：稱量完畢後，設定 Excel 游標自動往哪裡跳動（例如 down, right）。
  4. 防連發過濾時間：預設為 1 秒，用以過濾單次操作的冗餘訊號。
  5. 閒置斷線時間：預設為 30 分鐘。超時未操作會自動斷線並釋放連線。
💡 如何重新設定？
參數會儲存於同目錄的 config.json 中。每次啟動時會有 3 秒倒數，在此期間按下鍵盤上的 R 鍵，即可進入重新設定介面。

3. 日常作業流程

  1. 啟動程式：雙擊執行 ScaleMonitor.exe，確認顯示 Connected 與 Excel linked。
  2. 開啟 Excel：點選您想填入數據的起始儲存格。
  3. 天平稱量：在天平穩定後，按下天平上的 Print / 傳輸 鍵，數值即瞬間寫入 Excel 並自動跳格。
⚠️ 切換至 LIMS 紀錄系統注意事項
ScaleMonitor 運行時會「獨佔」天平的通訊埠。若您後續需要在 LIMS 系統中直接掃描或匯入重量，請先手動關閉 ScaleMonitor 視窗，釋放連線後，天平才能恢復純文字模式供 LIMS 抓取。

4. 設備相容性聲明

本軟體透過標準 RS232 進行數據接收，理論上支援大部分具備通訊功能的天平。目前已於以下廠牌與型號完成相容性驗證：

  • Mettler Toledo：XPE 205
  • OHAUS：AX224 (已針對此型號加入防連發過濾機制)
💡 相容性問題回報
因各廠牌天平的傳輸字串格式可能略有差異，若您在使用其他型號時遇到數據重複、漏字或無法識別的狀況，歡迎透過下方的信箱回報您的廠牌型號與狀況。

❓ 常見問題 (Q&A)

Q: 終端機一直顯示 [Excel Busy] 寫不進去？ A: 如果您正在 Excel 儲存格內「打字」（游標閃動的編輯狀態），Excel 會鎖定外部寫入。只要按下 Enter 鍵結束編輯狀態，天平數據就能正常寫入了。
Q: 程式突然自己關閉了？ A: 這通常是因為達到了您設定的「閒置時間」，系統為了節省資源並釋放連線所做的正常保護機制，重新執行程式即可。
